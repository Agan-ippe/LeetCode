# 分割链表

[原题链接](https://leetcode.cn/problems/partition-list-lcci/description/?envType=study-plan-v2&envId=cracking-the-coding-interview)

给你一个链表的头节点 `head` 和一个特定值 `x` ，请你对链表进行分隔，使得所有 **小于** `x` 的节点都出现在 **大于或等于** `x` 的节点之前。

你不需要 **保留** 每个分区中各节点的初始相对位置。

 

> **示例 1：**
>
> ![img](https://assets.leetcode.com/uploads/2021/01/04/partition.jpg)
>
> ~~~
> 输入：head = [1,4,3,2,5,2], x = 3
> 输出：[1,2,2,4,3,5]
> ~~~
>
> **示例 2：**
>
> ```
> 输入：head = [2,1], x = 2
> 输出：[1,2]
> ```

## 解题思路

~~~Java
/**
 * Definition for singly-linked list.
 * public class ListNode {
 *     int val;
 *     ListNode next;
 *     ListNode() {}
 *     ListNode(int val) { this.val = val; }
 *     ListNode(int val, ListNode next) { this.val = val; this.next = next; }
 * }
 */
class Solution {
    public ListNode partition(ListNode head, int x) {
        // 一次遍历
        // 所有小于x的节点，放在 small 链表中
        // 所有大于等于x的节点，放在 big 链表中
        // 拼接两个链表
        ListNode small =new ListNode(0);
        ListNode big = new ListNode(0);
        // 两个链表的头部节点
        ListNode sHead = small;
        ListNode bHead = big;
        // 节点分流，小的放小的，大的放大的
        while (head != null){
            if (head.val < x){
                sHead.next = head;
                sHead = sHead.next;
            } else {
                bHead.next = head;
                bHead = bHead.next;
            }
            head = head.next;
        }
        // 拼接
        sHead.next = big.next;
        bHead.next = null;
        return small.next;
    }
}
~~~
